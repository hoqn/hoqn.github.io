$$\begin{align}
A(n) &= (n-1) + \sum^{n-1}{i=0}{\frac{1}{n}(A(i) + A(n-1-i))} \
&= (n-1) + 2\sum^{n-1}{i=0}{\frac{1}{n}{A(i)}} \
&(\because A(1) = A(0) = 0)
\end{align}$$

참고) 수학적 귀납법(Induction)으로 증명

수학적 귀납법으로 임을 증명할 수 있다.

$$\begin{align}
A(n) &= (n-1) + 2\sum^{n-1}{i=0}{\frac{1}{n}A(i)} \
&\leq (n-1) + \frac{2}{n}\sum^{n-1}{i=0}{ci\ln(i)} \
&\leq (n-1) + \frac{2}{n}c\int^{n}_{1}{x\ln x}dx \
&\vdots\
A(n) &\leq 1.4n\lg n
\end{align}$$
